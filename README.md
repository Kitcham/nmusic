# Nmusic

---

## 一款可以试听网易云音乐,虾米音乐,QQ音乐的播放器.

功能很简单,搜索这三个平台的音乐,并试听.
这个项目是受到[Listen1](http://listen1.github.io/listen1/)的启发,并参考了部分的关于三个平台的网络接口.
